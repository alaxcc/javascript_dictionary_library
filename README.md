 ####Usage:
        init a instance.<br>
        var dic = new Dictionary();<br>
        add a key-value to dictionary<br>
        dic.add('a',123);<br>

        check if exist key in dictionary,here is return true<br>
        dic.contains('a')<br>

        get value by key<br>
        var v = dic.get('a')<br>

        remove the key-value from dictionary<br>
        dic.remove('a')<br>
